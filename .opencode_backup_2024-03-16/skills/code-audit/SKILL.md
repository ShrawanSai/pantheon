## What I do

- Performs a thorough analysis of the codebase, including Python backend, TypeScript frontend, and configuration.
- Identifies security vulnerabilities, dead code, poor implementations, and incomplete functionality.
- Assigns a severity score (1-5) to each finding (1 = low, 5 = critical/urgent).
- Provides actionable feedback, remediation strategies, and a detailed step-by-step mitigation plan for each finding from the perspective of a Senior Software Engineer.

## When to use me

Use this skill during code reviews, before major releases, or when planning refactoring sprints to ensure high code quality, security standards, and technical debt management.
